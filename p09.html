Project concept

This project is created based on the experimental camera i created last week. I wanted to continue with the collage style so that I choose to use the collage-sence font. To make it look more complete, I used switch() function to separate the starting interface and the camera following the template given by Munus. ot amke it look more playful and consistent, I make a gif to make it move.

Some thouhghts

For the buttons, in order to achieve better look, I decided to use p5.play library and using sprite to create better button that fits the style. Even though the styles can be chanegd with css, i still felt it pretty limited since i couldn 't really change the appearnce of that box and the fonts are also limietd compared to p5. js which allows for any font I uploaded.\n Besides, it is sort of suprising for me to find that consentful interface can be artistic if you spend time to design.
